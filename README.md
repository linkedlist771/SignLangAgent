# SignLangAgent

一个为聋哑人提供交互方便的agent。

---

*我之前遇到了一个很漂亮的女孩，可是她是聋哑人。为此，我想做一个聋哑人交流的多模态Agent。*

---
## Setup

This project uses [uv](https://docs.astral.sh/uv/) for dependency management.

### Install uv

```bash
pip install uv
```

### Install dependencies

```bash
uv sync
```

### Run

```bash
uv run main.py
```

### Add a dependency

```bash
uv add <package>
```
