# SignLangAgent

一个为聋哑人提供交互方便的agent。

An agent that provides convenient interaction for deaf-mute people.

## Setup

This project uses [uv](https://docs.astral.sh/uv/) for dependency management.

### Install uv

```bash
pip install uv
```

### Install dependencies

```bash
uv sync
```

### Run

```bash
uv run main.py
```

### Add a dependency

```bash
uv add <package>
```
